西南交通大学（SWJTU）与美国乔治亚州立大学（GSU）联合组队参加国际遗传工程机器大赛（iGEM）斩获银奖

2020-12-01 11:12:28.383   分类：要闻及动态   来源：教务处   阅读量：2752 返回列表


北京时间2020年11月23日凌晨，第十七届国际遗传工程机器竞赛（international Genetically Engineered Machine competition, iGEM，以下简称iGEM）正式落下帷幕，共有来自全球40多个国家的256支代表队（共计5000余名师生）参赛，包括来自清华、北大等国内C9院校，和哈佛、麻省理工、普林斯顿等13所常春藤院校。西南交通大学（Southwest Jiaotong University, SWJTU-China）生命科学与工程学院与美国乔治亚州立大学（Georgia State University, GSU-USA）组成了联合队伍“GA State SW Jiaotong“参赛，在长达一年的比赛周期中，团队克服了新冠疫情的不利因素，圆满地完成课题设计、研究工作、社会实践、网页制作、海报展示及答辩陈述等各环节，获得评委一致好评，斩获银奖。本项赛事由美国麻省理工学院（MIT）于2003年创办，每年一届，是培养生命学科未来科学家的摇篮，代表了本科阶段生命学科合成生物学领域竞赛的最高水平。




参加本次大赛的团队由15名西南交通大学学生（大部分来自生物工程中外合作办学项目）和8名乔治亚州立大学学生组成，由西南交通大学的黄新河、廖海、周嘉裕、朱乾坤等4位教师和乔治亚州立大学的Dr. Matt Brewer共同指导。该团队的部分教师和学生也参加了2019年第十六届iGEM比赛并获得铜奖。在吸纳新成员后，团队继续以“珊瑚白化研究（Synbio-dinium A synthetic biology solution to coral bleaching ）“为主题，采用合成生物学的涉及理念，构建转基因珊瑚共生藻，为珊瑚白化及灭绝这一世界难题提供了新的解决思路和方案。本次iGEM比赛采用线上答辩，评委组对小组成员进行全英文提问，同学们对答如流，充分展现了我校学生全面发展的综合素质。北京时间11月22日21:00，由黄新河、廖海老师带队，杨晨、李浩然、朱宁馨、谭国栋等4位学生代表前往北京iGEM分会场参加闭幕式。我校同学与来自清华、北大等其他院校的同学积极互动，深入交流学习比赛经验与心得。



自2019年末，团队便开始备战，寒假期间也花了大量时间泡在实验时研讨方案。在疫情期间同学们无法按时返校开展实验的情况下，师生们热情不减，线上讨论，分工合作，在家完成了课题设计，购置必要实验设备及材料试剂在家开展相关实验。在学生回到学校后，全身心进入实验室，抓紧一切时间完成实验内容，力争得到最好的实验结果。为了保证实验进度的正常开展，每周二晚上9点，中美双方的师生都会通过视频会议探讨实验进展和问题，交流实验心得和体会。12个小时的时差和太平洋的距离没有能阻拦两边同学们对科学研究似火的热情，加深了同学们的友谊与情谊。



在项目进行过程中，团队团结一致客服了重重困难，顺利完成项目。

本项目组计划采用基因枪转化的方法实现目标基因的成果转化，然而市面上基因枪价格昂贵，在预算本就吃紧的情况下购买成熟的基因枪设备显得捉襟见肘。但是同学们通过认真学习研究，利用文献资料，自主研发设计了一款基因枪。为了让基因枪性能更好，团队还专门请教了机械专业的同学，并一同前往交大创客空间进行开发创造。最终这款DIY基因枪不仅让实验顺利进行，也同时让同学们有机会通过生物学与其他学科的融合而丰富知识，锻炼能力。



在藻类基因转和藻类培养两个方面遇到问题后，项目组通过学院老师联系到厦门大学林昕教授，这位在研究珊瑚共生藻类上位于全国领先并具有突出贡献的专家给我校项目组的同学提出了宝贵的意见。当同学们问及利用基因工程技术改造虫黄藻、缓解珊瑚白化的前景怎样。林教授对我们的这一设想给予了高度肯定，对藻类研究过程提供了建设性意见，如培养藻类不宜过快，尤其要注意生态风险。此外，项目组还与来自四川大学的SCU-China iGEM团队进行了项目间的互相了解和讨论。在交流中，两校同学们相互学习，共同提高。



2020年10月16日，我校iGEM项目组成员前往西南交通大学附属中学进行题为《Save Coral Reefs》的宣讲，完成本次比赛要求的社会实践（Human Practice）环节。本次宣讲以主题演讲和话剧表演的形式，生动形象地向高中生展示为什么珊瑚会白化，我们是如何培养藻类、如何设计和制作基因抢并用它对生物进行基因改造的。同学们反响良好，互动积极。通过本次宣讲，同学们成功向社会普及了珊瑚白化相关知识，提高社会保护珊瑚的意识。



2020是特殊的一年，虽然因疫情不能前往波士顿同世界众高校交流学习，但“GA State SW Jiaotong“项目组的努力依然为本次参赛画上完美的句号。本届竞赛是两所学校第二次联合组队参赛，以西南交通大学与美国乔治亚州立大学合作举办生物工程专业本科教育项目为支撑，为学生提供了更广阔的科创平台，拓宽了学生的国际视野，对项目的建设起到了积极的推进作用。
